About This Project :-

This Weather App is a simple project built for learning purposes. It uses the Fetch API in JavaScript to request real-time weather data from a public weather API.

It helps beginners understand how to:

  • Use JavaScript Fetch API for making HTTP requests
  • Work with JSON data returned from APIs
  • Manipulate the DOM using JavaScript
  • Create a responsive UI using Bootstrap

? Why This Project?

The best way to learn is by building something small and useful. This app gives hands-on practice for students and new developers to connect front-end code with live data from the internet.
